Claude Code Prompt: Origin Labs Web Development
1. Project Identity & Core Tech
Project Name: Origin Labs Corporate Website

Framework: Next.js 14+ (App Router)

Language: TypeScript (Strict Mode)

Styling: Tailwind CSS

Animations: framer-motion & @studio-freight/react-lenis

2. Design System (Strict White Mode)
Theme: Pure Light Mode only. Remove all dark mode logic and theme toggles.

Background: White (#FFFFFF) with subtle section shifts to Off-White (#F9FAFB).

Primary Palette:

Main Gradient: Linear Gradient: #2DD4E0 (Cyan) to #0055FF (Deep Blue).

Accents: #1A73E8 (Blue) and #4B5563 (Slate Grey for text).

Typography:

Headings: Montserrat (Weights: 600, 700, 800).

Body: Inter (Weights: 400, 500).

UI Elements: Bento Grids (Apple-style), light glassmorphism (backdrop-blur), and soft layered shadows for depth.

3. Site Architecture (No i18n)
Structure: Flat hierarchy (No [lang] folders).

Pages:

Home: Hero with gradient/3D animation, Bento-Grid for services, process section.

Projects: Filterable grid for Websites, Webapps, and Mobile Apps.

Services: Detailed technical capabilities.

About: Team and vision.

Contact: Formspree/EmailJS integration & Calendly.

Legal: Imprint & Privacy.
